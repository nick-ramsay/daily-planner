The Daily Plans application is dependent upon ten packages...

 1. React
 2. Express
 3. Axios
 4. Mongoose
 5. JS-SHA256
 6. Moment.js
 7. if-env
 8. React-Spinners
 
If you have cloned the repository with the package.json file, these packages can be installed using the ```npm-install``` command.
